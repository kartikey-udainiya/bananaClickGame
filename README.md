Setup Guide 

Prerequisites:
1. Nodejs installed on the machine.
2. MongoDB

How to run the Project.
1. clone the project.
2. Install the dependencies "npm i"
3. Run the frontend "npm run dev"
4. Run the backend "npm run server"
